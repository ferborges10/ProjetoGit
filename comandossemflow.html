COMANDO FUNCIONALIDADE

git add .

Adciona todos os arquivos salvos para receber o commit

git commit

Comando para rotular e deixar os arquivos prontos para serem enviados para o github

git push

envia de fato os arquivos ao github

git tag

cria uma tag que marca e rotula as etapas do sistema

git checkout -b feature/nome-da-funcionalidade

comando para criar uma nova branch a partir da developer.

git push origin master --tags

comando para enviar todas as tags ao github

git clear

comando para limpar o painel de comandos
